# `UgolkiAI` #
Класс определяющий искусственный интеллект бота для игры в уголки.
model передает интеллекту ссылку на объект фрейма, [UgolkiAI](UgolkiAI.md) применяет свой алгоритм поиска приемлемого хода и испускает сигнал о его готовности. [UgolkiModel](UgolkiModel.md) перехватывает сигнал, изменяет положение игры и испускает сигнал, который в свою очередь перехватывает [UgolkiGUI](UgolkiGUI.md) для обновления интерфейса.
